# Dashboard Instagram

Proyecto estatico listo para subir a cualquier servidor web.

## Archivos

- `index.html`: dashboard one-page principal

## Publicacion

Sube el contenido completo de esta carpeta a tu servidor.

## Nota

Los posts destacados se muestran con embeds de Instagram dentro del dashboard.
Eso requiere que el navegador del visitante pueda cargar contenido externo desde Instagram.
